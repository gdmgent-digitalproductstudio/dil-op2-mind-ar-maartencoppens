# MindAR Face Detection Project

## Over dit project

Dit project heeft 2 kanten, een pagina waar het je gezicht scannt en je een uno speler maakt, en de andere functie is dat het programma 4 Markers kan herkennen (dit zijn uno kaarten) en hierbij een leuke sfeeranimatie toont dat te maken heeft met de uno kaart.

## Interacties

### Marker Switcher

- Klik op de **"Switch Card"** knop om willekeurig tussen vier verschillende markers te wisselen
- Met elke klik worden twee markers tegelijk gerandomiseerd
- Er klinkt een geluidssignaal ter bevestiging van de actie

### Hoe het werkt

De applicatie detecteert gezichtsfeatures en plaatst AR-markers op specifieke punten (zoals ogen). Door op de knop te klikken, worden deze markers willekeurig ververst en verschuift de AR-ervaring.

Daarnaast kan het ook 4 bepaalde uno kaarten herkennen en hierbij een leuke animatie tonen. De 4 markers staan in ./markers/, hier kan je ze uitprinten of downloaden en in een virtuele camera steken.
